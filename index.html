About Me

I am a student learning how to teach computational thinking through hands-on and reflective activities. I am interested in helping students build confidence with programming concepts.

Teaching Philosophy

I believe students learn best when they can experiment, make mistakes, and reflect on their thinking. Computational thinking should be accessible and engaging for all learners.

Project Videos

Events in Snap: Monkey and Banana Game

This video explains how I used events and broadcasts in Snap to create an interactive monkey and banana game. I describe how different sprites respond to events, how broadcasts allow sprites to communicate, and how changing event behaviors would affect the program. This demonstrates my understanding of event-driven programming and interaction between sprites.

Code explanation focusing on events and sprite interaction in Snap.

Nested Repeats in Snap: Mouse Octagon

This code explanation video demonstrates how nested repeat loops work in Snap. I explain how the outer repeat controls the overall pattern and how the inner repeat controls the smaller repeated movement steps. I also describe how changing the number of repeats changes the size and shape of the path.

Code explanation focusing on nested repeats and patterns.

Variables and Shapes in Snap

This video explains how I used variables in Snap to draw different shapes. I describe how variables control the number of sides and side length, how events start the program, and how changing variable values changes the shapes that are drawn. I also explain how variables in programming are different from variables in algebra.

Code explanation focusing on variables and shape drawing in Snap.

Conditional Loops in Snap: Catch the Soccer Ball

This video explains my Snap program where a soccer ball moves to random positions and the user tries to catch it with the mouse. I describe how the green flag event starts the program, how a repeat-until loop controls the movement, and how random x and y variables make the sprite move to unpredictable locations. This demonstrates my understanding of events, variables, and conditional loops working together.

Code explanation focusing on events and conditional loops in Snap.

Snap Target Game: If-Else Score Explanation

This video explains my Snap code where a blue ball moves randomly and the player clicks to score points. I describe how different colored areas change the score using if and if-else logic, and how a change in those conditions would change the behavior of the program.

Code explanation for my Snap target game focusing on conditional logic.

Nested If-Else in Snap: Norman the Penguin Adventure

This video explains my Snap program that demonstrates how nested if-else statements work. In this project, Norman the Penguin decides whether he can go swimming in the ocean based on two conditions: whether he knows how to swim and whether he is old enough. I walk through how the outer if-else checks the user's input and how the inner (nested) if-else checks the age variable. I also explain how changing the conditions would affect the program’s outcome.

Code explanation focusing on nested if-else decision making in Snap.
